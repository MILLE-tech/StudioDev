Banque d'exercice JS



Exercices disponibles :

Exercice 1 : Trouver le plus grand

Créer une fonction qui retourne le plus grand parmi trois nombres donnés.




Exercice 2 : Table de multiplication

Générer la table de multiplication d’un nombre donné (1 à 10).




Exercice 3 : Calculer la moyenne

Entrer une liste de nombres et retourner leur moyenne.




Exercice 4 : Vérifier année bissextile

Créer une fonction qui vérifie si une année donnée est bissextile.




Exercice 5 : Convertir Celsius en Fahrenheit

Créer une fonction qui convertit une température en Celsius en Fahrenheit.




Exercice 6 : Supprimer les doublons

Créer une fonction qui retire les doublons d’un tableau.




Exercice 7 : Trier un tableau

Créer une fonction qui trie un tableau de nombres en ordre croissant.




Exercice 8 : Générer un nombre aléatoire

Créer une fonction qui génère un nombre aléatoire entre deux bornes définies.




Exercice 9 : Compter les caractères uniques

Créer une fonction qui compte le nombre de caractères uniques dans une chaîne.




Exercice 10 : Trouver le deuxième plus grand

Créer une fonction qui trouve le deuxième plus grand nombre dans un tableau.




Exercice 11 : Compter les mots

Créer une fonction qui compte le nombre de mots dans une phrase donnée.




Exercice 12 : Inverser une phrase

Créer une fonction qui inverse complètement une phrase donnée.




Exercice 13 : Générer un mot de passe

Créer une fonction qui génère un mot de passe aléatoire de longueur définie.




Exercice 14 : Vérifier une anagramme

Créer une fonction qui vérifie si deux chaînes sont des anagrammes.




Exercice 15 : Vérifier un carré parfait

Créer une fonction qui vérifie si un nombre est un carré parfait.
